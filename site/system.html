The Aircloak Anonymized Analytics System:
User Anonymity without Data Loss

Aircloak Analytics is the first system to combine high-accuracy analytics with strong anonymity.
It brings together a unique blend of advanced technology along with unprecedented transparency and
oversight. Aircloak Analytics frees up user data, enabling new use cases and ways to monetize.

Cloaked Analytics Data Engine

At the core of all Aircloak solutions is the Cloaked Analytics Data Engine, or "Cloak" for short. The Cloak utilizes "Trusted Computing" secure hardware and Security Enhanced Linux (SELinux). The result is a password-free secure system with enforced oversight and cryptographically proven transparency.


Cloak Deployments

Aircloak’s Cloaked Analytics Data Engine can be deployed to satisfy a wide variety of business use cases. Some of them "just" make current tasks easier, cheaper and more secure, but we are even more passionate about the ones that open up a whole world of new possibilities.

Data collection from user devices

User data transmitted from user device to Cloak is fully protected the moment it leaves the user device. Application developers can obtain rich aggregate user analytics without concern over privacy loss.

Data sharing within organizations

Organizations can safely transfer user data to cloaks either inside or outside the firewall perimeter. Analytics over user data can be made freely available to all groups within the organization.

Data sharing with partner organizations

Organizations may safely provide analytics over user data to partner organizations. This paves the way for new forms of data monetization.

Data brokering between organizations

To fully unlock the value of user data, multiple organizations can join their data together inside cloaks while retaining all anonymity properties.
